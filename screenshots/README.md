# Скриншоты приложения

## 01. Главная страница
**Файл:** `01-homepage.png`

**Описание:**
- Поиск стажёрских ML-вакансий
- Интеграция 3 источников (Habr Career, hh.ru, Сбербанк)
- Фильтр по уровню: Intern / Junior
- Тёмная тема, Tailwind CSS

**Как сделать скриншот:**
1. Запустить сервер: `uvicorn app:app --reload`
2. Открыть http://localhost:8000
3. Скриншот всего окна браузера (1920x1080)

## 02. Результаты поиска
**Файл:** `02-vacancies.png`

**Описание:**
- Список вакансий из всех источников
- Каждая карточка: название, компания, зарплата, локация
- Кнопка "✦ Откликнуться" на каждой вакансии
- Фильтры сверху: Стажёр | Junior | Все

**Как сделать скриншот:**
1. В поиске ввести "ML"
2. Нажать "Найти"
3. Скриншот результатов (1920x1080)

## Требования к скриншотам

- **Формат:** PNG или JPEG
- **Разрешение:** 1920x1080 (горизонтальные), 4:3 (профиль)
- **Тема:** Тёмная (dark mode)
- **Качество:** Высокое, читаемый текст
- **Количество:** 6+ скриншотов для README
